Instructions

  1. Switch between languages to see how missing translations behave
  2. Look at the browser console for detailed logging
  3. Click "Toggle Missing Translation Display Mode" to show missing keys directly
  4. Click "Find All Missing Translations" to get a full report
  5. Hover over elements with missing translations to see tooltips

Console Commands

You can also run these commands in the browser console:

// Toggle between showing missing keys vs fallback text
toggleMissingTranslationDisplay()

// Find all missing translations for all languages
findMissingTranslations()

// Find missing translations for a specific language
findMissingTranslations('nl')

// Manually update content to a specific language
updateContent('en')
