          // JavaScript that dynamically calculates and displays my age
          function calculateAge(birthday) {
            const today = new Date();
            const birthDate = new Date(birthday);
            let age = today.getFullYear() - birthDate.getFullYear();
            const monthDifference = today.getMonth() - birthDate.getMonth();
            const dayDifference = today.getDate() - birthDate.getDate();

            // Adjust age if today's date is before the birthday this year
            if (monthDifference < 0 || (monthDifference === 0 && dayDifference < 0)) {
              age--;
            }

            return age;
        }

Language Support

This website supports English and Dutch, using a lightweight custom translation system that updates all text instantly via data-i18n attributes. No page reloads.

  • Active language is stored in localStorage
  • Missing keys fall back to English and are highlighted for debugging
  • Translations are structured and nested for modularity
