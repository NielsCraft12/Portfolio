WA-LCH Origins

Description

Made in ~30 hours during a game jam (theme: “”Bubbles”) — built with Unity and C#. WA-LCH Origins is a chaotic mini-cleanup adventure where you play as a janitor scrubbing away trash and trapping rogue rats using bubble-based tools. Built entirely under game jam pressure, it features player-controlled physics movement, animated doors, trap-based rat removal, and a bubbly cleaning mechanic.

What I made

Player Movement

Handled via Rigidbody for responsive physics-based control. Rotation matches movement direction for an intuitive feel.


  rb.linearVelocity = new Vector3(
    moveDirection.y * maxSpeed,
    yVelocity,
    moveDirection.x * maxSpeed
);

Cleaning mechanic

When activated, the cleaning ability shows a particle effect and wipes nearby trash or rats from the scene. Using an OverlapSphere to detect the trash


Collider[] hitColliders = Physics.OverlapSphere(
    cleaningpoint.transform.position,
    cleaningRadius
);

Trap Launching

The player shoots traps on cooldown to deal with fast-moving rats.


        if (trapcooldownTimer <= 0)
        {
           Instantiate(trap, trapSpawnPoint.position, trapSpawnPoint.rotation);
           trapcooldownTimer = trapcooldown;
        }

Door System

Doors open and close automatically when the player enters a trigger zone. Managed with a coroutine-like update loop in.


          if (!other.gameObject.GetComponent().Moving)
            {
                other.gameObject.GetComponent().Moving = true;
            }
