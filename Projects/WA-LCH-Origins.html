WA-LCH Origins

Description

Made in ~30 hours during a game jam (theme: “Bubbles”). Built with Unity and C#. WA-LCH Origins is a chaotic mini-cleanup adventure where you play as a janitor robot scrubbing away trash and trapping rogue rats using bubble-based tools. Built entirely under game jam pressure, it features player-controlled physics movement, animated doors, trap-based rat removal, and a bubbly cleaning mechanic.

What I Made

Player Movement

Handled via Rigidbody for responsive physics-based control. Rotation matches movement direction for an intuitive feel.

Doors

Doors open and close automatically when the when player or npc's enters a trigger zone. It tells the door to change the door state to open.

View on GitHub 

            if (other.gameObject.CompareTag("Door"))
         {
            if (other.gameObject.GetComponent().Moving == false)
            {
                other.gameObject.GetComponent().Moving = true;
            }
         }

The door then moves to the open position and waits for a certain amount of time before closing again.

View on GitHub 

        void MoveDoor(Vector3 goalPos)
        {
            float dist = Vector3.Distance(transform.position, goalPos);
            if (dist >= 0.001f)
            {
                transform.position = Vector3.MoveTowards(
                    transform.position,
                    goalPos,
                    speed * Time.deltaTime
                );
            }
            else
            {
                if (opening)
                {
                    delay += Time.deltaTime;
                    if (delay > openTime)
                    {
                        opening = false;
                        delay = 0.0f;
                    }
                }
            }
        }
