Slimetastic Punchout

Description

Slimetastic Punchout is a chaotic and fun local multiplayer game where you take control of colorful, squishy slimes and battle it out in fast-paced arenas. The goal? Cover as many tiles as you can while sliding, bouncing, and splattering your way across the map. Along the way, you can land some solid punches to shove your opponents out of the way or just throw them off their game. Every match is unpredictable, full of laughter, and a messy fight for control. It’s slimy chaos at its best!

What I Made

Custom Player Join System

I built a join system from scratch, allowing players to:

  • Dynamically join the game
  • Customize slime color and hats
  • Indicate when they're ready
  • Prevent duplicate colors via a locking system

How it works:

Players spawn into the join UI and navigate using directional inputs. Here’s a snippet of the navigation system:

public void NavigateMenu(InputAction.CallbackContext context)
{
    Vector2 moveDir = context.ReadValue<Vector2>();
    if (moveDir == new Vector2(0, 1)) { selected++; }
    else if (moveDir == new Vector2(0, -1)) { selected--; }
    
    rectTransform.localPosition = PlayerMenuChilds[selected].transform.localPosition;
    UpdateReadyText();
}

Color Selection & Locking System

Each player picks a unique color from a shared palette. If a color is taken, a lock icon appears and the player is prevented from readying up with that color.

View on GitHub 

          public void CheckIfColorUsed()
{
    foreach (ColorTracker tracker in menuManager.takenColors)
    {
        if (tracker.Color == colorSelected && tracker.PlayerName != transform.parent.name)
        {
            // Lock the color
            shouldShowLock = true;
        }
    }
    Locks.ForEach(lockIcon => lockIcon.GetComponent<Image>().enabled = shouldShowLock);
}

Player Movement

Each slime uses a physics-based movement system with:

  • Full Rigidbody control
  • Smooth rotation toward velocity
  • Jumping with air control
  • Movement dampening when no input is given

This snippet preserves vertical velocity while allowing grounded or aerial movement with different strengths.
